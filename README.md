# Dockerized GraphQL

Simple Post API for using GraphQL with Docker.

## Setup

* Create venv
```
py -m venv venv
```
* Activate venv
```
venv\Scripts\activate
```
* Install requirements.txt
```
pip install -r requirements.txt
```
